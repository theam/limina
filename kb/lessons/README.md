# Lessons

Keep reusable lessons here as small topic files.

Rules:

- Do not turn this directory into a running ledger.
- Prefer one file per topic or recurring failure mode.
- Keep each file short and concrete.
- Read only the lesson files relevant to the current problem.
- Link back to the artifact that produced the lesson when possible.

Good examples:

- `evaluation-trust.md`
- `query-understanding-failures.md`
- `gpu-benchmarking.md`

Bad examples:

- `all-lessons-ever.md`
- a giant chronological dump
